--- title: Dropping Single-Value Columns in Pandas author: Eric Moyer date: '2018-08-15' tags: - Python - Pandas - Data Wrangling slug: dropping-single-value-columns-in-pandas ---

Sometimes as you boil your data down, you end up with columns that either contain no data at all (or very little) or contain the same value in every row. Columns like this are all noise and no signal, and I generally like to get rid of them.

Dropping empty or nearly empty columns is easy, but what about columns that contain the same non-NaN value everywhere? This Jupyter notebook shows one way to do it.
